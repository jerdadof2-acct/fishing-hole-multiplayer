# Required Textures

This directory should contain the following texture files:

## Required Files

1. **waterNormals1.jpg** - Seamless water normal map texture (512x512 or 1024x1024 recommended)
   - Used for water surface distortion/animation
   - Must be tileable/seamless
   - Format: JPG normal map

2. **waterNormals2.jpg** - Second seamless water normal map texture (different pattern from #1)
   - Used for dual-scrolling normal map effect
   - Must be tileable/seamless  
   - Format: JPG normal map

3. **particle.png** - Small circular white particle sprite (64x64 or 128x128)
   - Used for floating water particles
   - Should be a white circle with soft edges
   - Format: PNG with transparency

## Where to Get These Textures

### Water Normal Maps:
1. **Poly Haven** (free): https://polyhaven.com/textures
   - Search for "water normal" or "ocean normal"
   - Download seamless normal maps

2. **CGTextures/Textures.com** (free account available): https://www.textures.com
   - Search for "water normal map"
   - Look for seamless/tileable options

3. **FreePBR** (free): https://freepbr.com
   - Search water-related materials
   - Download normal maps

4. **Create your own** using tools like:
   - Substance Designer
   - Photoshop with filters
   - Online normal map generators

### Particle Sprite:
- Can create in any image editor (Photoshop, GIMP, etc.)
- Simple white circle on transparent background
- Or use online sprite generators

## Note
If these files are missing, the application will automatically generate procedural fallbacks, but actual textures will look much better!
